# README pour NEXT.JS

Voici un README d'exemple pour la partie installation et développement d'un projet sous NEXT.JS.

## Installation

1.  **Clonage du dépôt :**

Dans un terminal, entrez la commandes suivante pour télécharger localement le repository :

```bash
git clone https://github.com/Azarta2Zygoto/visualization-wood.git
```

Pour le reste de l'installation, déplacez vous dans le dossier qui a été généré :

```bash
cd visualization-wood
```

2.  **Installation des dépendances :**

Il est recommandé d'avoir [npm](https://nodejs.org/en/download/) installé sur son ordinateur ! Sinon, il est possible d'utiliser un autre package manager comme [yarn](https://yarnpkg.com/getting-started/install), [bun](https://bun.com/docs/installation) ou [pnpm](https://pnpm.io/installation).

Dans un terminal, entrez :

```bash
# Pour npm
npm install
# Pour yarn
yarn install
# Pour bun
bun install
# Pour pnpm
pnpm install
```

## Lancement

Pour lancer l'application en mode développement, il vous suffit d'écrire dans un terminal dédié :

```bash
# Pour npm
npm run dev
# Pour yarn
yarn dev
# Pour bun
bun dev
# Pour pnpm
pnpm dev
```

Vous devriez alors avoir comme deux derniers messages dans le terminal :

```bash
 ✓ Starting...
 ✓ Ready in X.Xs
```

De plus, l'application tourne par défaut sur le port 3000 du localhost. L'url de l'application est donc à : http://localhost:3000/visualization-wood
